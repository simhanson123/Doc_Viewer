## Installation Windows

Téléchargez l'installateur ou le portable depuis Releases. Ouvrir ou Ctrl+O.

Supported formats: **MD · TXT · PDF · EPUB · DOCX**

## Développement

```bash
npm install
npm run dev
npm run electron:build:win
```

UI language: **Paramètres → Langue** (20 locales).
