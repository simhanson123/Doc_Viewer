## Instalación en Windows

Descargue el instalador o portable desde Releases. Use Abrir o Ctrl+O.

Supported formats: **MD · TXT · PDF · EPUB · DOCX**

## Desarrollo

```bash
npm install
npm run dev
npm run electron:build:win
```

UI language: **Ajustes → Idioma** (20 locales).
