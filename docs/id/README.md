## Instalasi Windows

Unduh installer atau portable dari Releases. Buka atau Ctrl+O.

Supported formats: **MD · TXT · PDF · EPUB · DOCX**

## Pengembangan

```bash
npm install
npm run dev
npm run electron:build:win
```

UI language: **Pengaturan → Bahasa** (20 locales).
