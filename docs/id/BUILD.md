# Onjeom — Build (Bahasa Indonesia)

## Windows

```bash
npm install
npm run electron:build:win
```

Keluaran di release/. Dev: npm run dev.

## Linux / Android

Linux/Android: lihat BUILD bahasa Inggris.

```bash
npm run electron:build:linux
npm run android:sync
```
