## Встановлення Windows

Завантажте інсталятор або portable з Releases. Відкрити або Ctrl+O.

Supported formats: **MD · TXT · PDF · EPUB · DOCX**

## Розробка

```bash
npm install
npm run dev
npm run electron:build:win
```

UI language: **Параметри → Мова** (20 locales).
