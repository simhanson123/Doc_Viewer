# Ончом (Onjeom) — Посібник користувача (Українська)

## Відкрити документ

Кнопка «Відкрити», Ctrl+O або перетягування. Формати: .md .txt .pdf .epub .docx.

## Читання та позначки

Режими одна/розворот/прокрутка/перекомпонування. Інструменти: перо, маркер, фігури, стікери.

## Експорт

PDF з позначками (Ctrl+E), PNG, JSON.
