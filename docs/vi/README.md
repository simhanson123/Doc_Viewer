## Cài đặt Windows

Tải bản cài hoặc portable từ Releases. Mở hoặc Ctrl+O.

Supported formats: **MD · TXT · PDF · EPUB · DOCX**

## Phát triển

```bash
npm install
npm run dev
npm run electron:build:win
```

UI language: **Cài đặt → Ngôn ngữ** (20 locales).
