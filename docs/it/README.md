## Installazione Windows

Scarica installer o portable da Releases. Apri o Ctrl+O.

Supported formats: **MD · TXT · PDF · EPUB · DOCX**

## Sviluppo

```bash
npm install
npm run dev
npm run electron:build:win
```

UI language: **Impostazioni → Lingua** (20 locales).
