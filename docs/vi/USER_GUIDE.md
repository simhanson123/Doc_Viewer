# Onjeom — Hướng dẫn sử dụng (Tiếng Việt)

## Mở tài liệu

Nút Mở, Ctrl+O hoặc kéo thả. Định dạng: .md .txt .pdf .epub .docx.

## Đọc và chú thích

Chế độ một trang/hai trang/cuộn/chảy lại. Công cụ: bút, đánh dấu, hình, ghi chú.

## Xuất

PDF có chú thích (Ctrl+E), PNG, JSON.
