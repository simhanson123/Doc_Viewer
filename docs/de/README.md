## Windows-Installation

Installer oder Portable von Releases herunterladen. Öffnen oder Strg+O.

Supported formats: **MD · TXT · PDF · EPUB · DOCX**

## Entwicklung

```bash
npm install
npm run dev
npm run electron:build:win
```

UI language: **Einstellungen → Sprache** (20 locales).
