## ติดตั้ง Windows

ดาวน์โหลดตัวติดตั้งหรือ portable จาก Releases แล้วกดเปิดหรือ Ctrl+O

Supported formats: **MD · TXT · PDF · EPUB · DOCX**

## การพัฒนา

```bash
npm install
npm run dev
npm run electron:build:win
```

UI language: **การตั้งค่า → ภาษา** (20 locales).
