## Windows-installatie

Download installer of portable van Releases. Openen of Ctrl+O.

Supported formats: **MD · TXT · PDF · EPUB · DOCX**

## Ontwikkeling

```bash
npm install
npm run dev
npm run electron:build:win
```

UI language: **Instellingen → Taal** (20 locales).
