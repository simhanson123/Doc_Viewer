## Windows 安裝

從 Releases 下載安裝檔或可攜版，以「開啟」或 Ctrl+O 開啟文件。

Supported formats: **MD · TXT · PDF · EPUB · DOCX**

## 開發

```bash
npm install
npm run dev
npm run electron:build:win
```

UI language: **設定 → 語言** (20 locales).
