# 온점 — 사용 설명서 (한국어)

## 문서 열기

- **문서 열기** 버튼 또는 `Ctrl+O`
- 창에 파일을 드래그 앤 드롭
- 지원: `.md` `.txt` `.pdf` `.epub` `.docx`

왼쪽 **서재**에서 문서를 전환하고, 검색·형식·폴더로 정리할 수 있습니다.

## 읽기

| 조작 | 설명 |
|------|------|
| 낱장 / 펼침 / 스크롤 / 리플로우 | 보기 모드 |
| `−` / `%` / `+` | 확대·축소 (% 클릭 시 폭 맞춤) |
| `←` `→` | 이전·다음 페이지 |
| `G` | 페이지 번호 이동 |
| 리본 / `B` | 책갈피 |
| 우측 패널 / `T` | 목차·검색·하이라이트·메모 |

**리플로우**는 긴 글 읽기에, **낱장·펼침**은 필기에 적합합니다.

## 필기 도구

| 도구 | 용도 |
|------|------|
| 선택 | 텍스트 선택 |
| A (텍스트 하이라이트) | 문장 클릭 하이라이트 |
| 형광펜 / 펜 | 자유 필기 (스타일러스 필압 지원) |
| 직선 · 도형 · 지우개 · 노트 · 레이저 | 추가 도구 |
| 실행 취소 / 다시 실행 | `Ctrl+Z` / `Ctrl+Shift+Z` |

필압 곡선은 **설정**에서 바꿀 수 있습니다.

## 내보내기 · 동기화

- 필기 포함 PDF (`Ctrl+E`)
- 현재 쪽 PNG
- 필기 JSON 내보내기/가져오기
- 데스크톱: 동기화 폴더에 `.onjeom.json` 자동 저장

필기는 이 기기에도 자동 저장됩니다.

## 단축키

| 단축키 | 동작 |
|--------|------|
| `Ctrl+O` | 문서 열기 |
| `Ctrl+E` | 필기 PDF |
| `Ctrl+,` | 설정 |
| `Ctrl+/` | 단축키 도움말 |
| `B` | 책갈피 |
| `F` | 서재 검색 |
| `1`–`9` | 도구 선택 |
