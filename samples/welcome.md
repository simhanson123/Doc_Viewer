# 온점에 오신 것을 환영합니다

이 파일은 샘플 마크다운입니다. **문서 열기**로 실제 MD / PDF / EPUB / DOCX 를 불러올 수 있습니다.

## 필기 도구

하단 도구 모음에서 다음을 사용할 수 있습니다.

1. 선택 — 텍스트 드래그
2. 가 — 문장 하이라이트
3. 형광펜 / 펜 — 자유 필기
4. 지우개 · 도형 · 스티키 노트

## 코드 블록

```ts
const page = {
  content: '본문',
  ink: 'SVG 필기 레이어',
  notes: '스티키 노트',
};
```
