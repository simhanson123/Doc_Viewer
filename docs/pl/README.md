## Instalacja Windows

Pobierz instalator lub portable z Releases. Otwórz lub Ctrl+O.

Supported formats: **MD · TXT · PDF · EPUB · DOCX**

## Rozwój

```bash
npm install
npm run dev
npm run electron:build:win
```

UI language: **Ustawienia → Język** (20 locales).
