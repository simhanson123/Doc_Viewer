## تثبيت Windows

حمّل المثبت أو النسخة المحمولة من Releases. افتح أو Ctrl+O.

Supported formats: **MD · TXT · PDF · EPUB · DOCX**

## التطوير

```bash
npm install
npm run dev
npm run electron:build:win
```

UI language: **الإعدادات → اللغة** (20 locales).
