## Windows इंस्टॉल

Releases से इंस्टॉलर या portable डाउनलोड करें। खोलें या Ctrl+O।

Supported formats: **MD · TXT · PDF · EPUB · DOCX**

## विकास

```bash
npm install
npm run dev
npm run electron:build:win
```

UI language: **सेटिंग्स → भाषा** (20 locales).
