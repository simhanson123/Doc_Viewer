# 句点 — 使用指南（简体中文）

## 打开文档

- **打开** 或 `Ctrl+O`
- 拖放到窗口
- 支持：`.md` `.txt` `.pdf` `.epub` `.docx`

## 阅读与批注

单页 / 双页 / 滚动 / 重排；缩放与翻页；`B` 书签。  
底部工具栏：选择、文本高亮、荧光笔、钢笔、直线、橡皮、图形、便签、激光、撤销/重做。

**重排**适合长文阅读；**单页/双页**适合书写。

## 导出

带批注 PDF（`Ctrl+E`）、PNG、批注 JSON；桌面版可设置同步文件夹。
