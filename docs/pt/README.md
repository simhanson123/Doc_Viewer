## Instalação no Windows

Descarregue o instalador ou portable em Releases. Use Abrir ou Ctrl+O.

Supported formats: **MD · TXT · PDF · EPUB · DOCX**

## Desenvolvimento

```bash
npm install
npm run dev
npm run electron:build:win
```

UI language: **Definições → Idioma** (20 locales).
