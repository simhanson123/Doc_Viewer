# Ончом (Onjeom) — Русский

Многоформатный просмотрщик документов с рукописными пометками.  
**Лицензия:** MIT · **Repo:** [simhanson123/Doc_Viewer](https://github.com/simhanson123/Doc_Viewer)

- [Руководство пользователя](./USER_GUIDE.md)

- [All languages](../README.md)

## Установка Windows

Скачайте установщик или portable с Releases. Открыть или Ctrl+O.

Supported formats: **MD · TXT · PDF · EPUB · DOCX**

## Разработка

```bash
npm install
npm run dev
npm run electron:build:win
```

UI language: **Настройки → Язык** (20 locales).
